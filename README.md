# RCMS
餐厅收银管理系统<br>  
==
是用于餐厅的收银管理系统，包含了四个模块<br>  
1.桌位模块<br>  
---
  桌位模块主要是用于管理桌位的模块，包括点菜到结账的流程<br>  
2.账单模块<br>  
---
  账单模块记录了每一天的帐单汇总，同时提供了年月日账单的统计，在日账单内可以查看当日的所有消费详情，还提供了按日期或日期区间搜索账单的功能<br>  
3.日常维护模块<br>  
--
 提供了桌位菜单用户供应商的配置功能，也就是增删改查<br>  
4.酒水库存模块<br>  
 --
 查看添加酒水的进销存信息<br>
